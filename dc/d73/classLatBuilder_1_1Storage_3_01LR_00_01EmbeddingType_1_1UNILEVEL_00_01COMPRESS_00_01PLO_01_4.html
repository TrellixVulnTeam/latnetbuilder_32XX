Detailed Description

template<LatticeType LR, Compress COMPRESS, PerLevelOrder PLO>
class LatBuilder::Storage< LR, EmbeddingType::UNILEVEL, COMPRESS, PLO >

Storage class for flat vectors.

The vector elements are not permuted but compression is applied to vector indices.

Template Parameters
COMPRESSCompression type (either None or Symmetric).
