Detailed Description

template<Compress COMPRESS>
class LatBuilder::StorageTraits< Storage< LatticeType::DIGITAL, EmbeddingType::UNILEVEL, COMPRESS > >::Stride

Stride permutation.

For digital nets: a Stride with takes as parameter a generating matrix \(M\) is the mapping that maps an index \(i\) as follows:

  • convert the binary representation of \(i\) to its associated Gray code \(\boldsymbol i'\)
  • compute \(M \boldsymbol i'\) which is the reversed binary representation of the result integer

Since the integers are first converted to Gray code, the computation of \(M \boldsymbol i'\) can be fastened as explained in these notes by Joe and Kuo on the generation of Sobol' sequences.
