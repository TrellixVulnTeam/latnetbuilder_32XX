Detailed Description

One-dimensional merit function for the \(\mathcal{P}_{\alpha,PLR}\) discrepancy.

This merit function is defined as

\[ \omega(x) = \mu(\alpha) - 2^{(1+ \lfloor \log_{2}(x_{i,j}) \rfloor)(\alpha-1)}(\mu(\alpha)+1) \]

for \(\alpha >1\)
